6. IMPLEMENTATION PHASES (When Ready)
--------------------------------------------------------------------------------

Phase 1 - UI Component
  - Create Chatbot.jsx component
  - Implement layout: greeting, 5 suggested buttons, input + send button
  - Floating widget (bottom-right) or inline section
  - Match reference: clean, minimal, pill-shaped buttons, rounded input

Phase 2 - Data Layer
  - Create chatbot keyword-answer config (JSON or JS module)
  - Populate with all extracted keywords and answers from Section 3
  - Optional: Fetch active jobs from ApiService for "Careers" responses

Phase 3 - Chat Logic
  - Keyword matching (simple: includes, or use fuzzy matching library)
  - Button click -> pre-fill query or show instant answer
  - Free-form input -> search keywords -> return best match
  - Chat history display (optional)

Phase 4 - Polish
  - Scroll to section (e.g., "Contact" -> smooth scroll to #contact)
  - Responsive design
  - Open/close toggle animation
  - Accessibility (keyboard, screen readers)
